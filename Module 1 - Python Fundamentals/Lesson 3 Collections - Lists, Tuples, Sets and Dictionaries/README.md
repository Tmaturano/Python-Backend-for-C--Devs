# Lesson 3 - Collections: Lists, Tuples, Sets and Dictionaries

## Overview

This lesson covers Python's built-in collection types, which are essential for storing and organizing multiple values. These collections are more versatile than C# arrays and represent one of Python's greatest strengths.

## Topics Covered

- **Lists**: Ordered, mutable collections of items
- **Tuples**: Ordered, immutable collections of items
- **Sets**: Unordered, unique, mutable collections
- **Dictionaries**: Key-value pairs (similar to C# dictionaries)

## Files

- `list.py` - List creation, manipulation, and iteration
- `tuple.py` - Tuple usage, unpacking, and immutability
- `set.py` - Set operations, membership testing, and unique values
- `dictionary.py` - Dictionary operations, keys, values, and common patterns

## Key Takeaway

Mastering collections is crucial for Python development. These data structures form the backbone of most Python programs and are more Pythonic than manually managing arrays or custom collection classes.
