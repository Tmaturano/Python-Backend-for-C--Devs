# Lesson 2 - Functions and Modules

## Overview

This lesson teaches you how to organize code through functions and modules, enabling reusability and modularity—key principles for professional Python development.

## Topics Covered

- **Functions**: Defining functions, parameters, return values, and function scope
- **Modules**: Creating and importing modules to organize code
- **Code Organization**: Structuring Python projects with multiple files
- **Function Best Practices**: Writing maintainable, reusable functions

## Files

- `functions.py` - Function definition and usage patterns
- `calculator.py` - Practical example: Calculator module with mathematical functions
- `main.py` - Importing and using modules together

## Key Takeaway

After this lesson, you'll be able to write modular, reusable code and understand how to organize Python projects beyond single files. This mirrors the module/namespace concepts you know from C#.
