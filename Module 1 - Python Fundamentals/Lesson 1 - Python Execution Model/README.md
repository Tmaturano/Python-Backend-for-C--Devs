# Lesson 1 - Python Execution Model

## Overview

This lesson covers the fundamentals of how Python code executes and the core data types and operators you'll use daily.

## Topics Covered

- **Python Types**: Understanding Python's dynamic type system and built-in types
- **Strings**: String manipulation, formatting, and common string operations
- **Boolean Operators**: Logical operations (`and`, `or`, `not`)
- **Comparison Operators**: Comparing values with `==`, `!=`, `<`, `>`, `<=`, `>=`
- **Conditions**: Control flow using `if`, `elif`, and `else` statements

## Files

- `python_types.py` - Introduction to Python's type system
- `strings.py` - String handling and manipulation
- `boolean_operators.py` - Logical boolean operations
- `comparison_operators.py` - Value comparison operations
- `conditions.py` - Conditional control flow statements
- `lesson1.py` - Comprehensive lesson overview
- `homework/user_profile.py` - Practice exercise: Building a user profile

## Key Takeaway

This lesson establishes the foundation for Python programming by teaching you how Python executes code and the fundamental building blocks you'll use in every program.
